# Adivinhe o Número

## Descrição

Este é um simples jogo baseado na web chamado "Adivinhe o Número". O jogo gera um número aleatório entre 1 e 100, e o objetivo do jogador é adivinhar o número correto. Após cada palpite, o jogador recebe feedback indicando se o palpite está muito alto, muito baixo ou correto. O jogo também limita o número de palpites permitidos e fornece feedback visual ao jogador.

## Como Jogar

1. Abra o arquivo `index.html` em seu navegador da web.
2. Leia as instruções fornecidas na página.
3. Insira seu palpite no campo de entrada fornecido.
4. Clique no botão "Verificar" para verificar seu palpite.
5. Receba feedback sobre seu palpite:
   - Barra verde: Seu palpite está "frio" e o número real é maior.
   - Barra amarela: Seu palpite está "quente" e o número real é menor.
   - Barra preta: Parabéns! Você acertou o número.
   - Barra vermelha: O jogo acabou. Você atingiu o limite de palpites sem acertar o número correto.
6. Repita os passos 3-5 até acertar o número correto ou atingir o limite de palpites.

## Recursos

- Gera um número aleatório entre 1 e 100.
- Fornece feedback visual ao jogador com base em seus palpites.
- Limita o número de palpites permitidos.
- Suporta validação de entrada para garantir palpites válidos.
- Exibe palpites anteriores para referência.
- Interface fácil de entender com instruções claras.
